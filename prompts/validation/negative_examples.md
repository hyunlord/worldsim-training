# Negative Example Guide

- 반복 루프: `이 사람은 이다. 이 사람은 이다.`
- 한자어 과다 사용: `식량을 수집했다.`
- 어투 혼용: `위험하오... 빨리 뛰어!`
- 길이 초과: 5문장 이상 장황한 출력
- 현대어 누출: `OK, 알겠어.`
- Layer 3 JSON 파손: 닫는 중괄호 누락, 키 이름 오타, 문자열 대신 설명문 출력
- Task E 잘못된 선택: `action_id`가 선택지 범위를 벗어남
- Task F 잘못된 감정: `emotion`이 8개 허용값 밖이거나 `cause`가 빈 문자열
